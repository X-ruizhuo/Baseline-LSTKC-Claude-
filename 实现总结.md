## 核心创新点

### 1. 长短期知识解耦 ⭐⭐⭐
将旧模型的知识分解为:
- **长期知识**: 域不变的通用特征 (强约束 2.0x)
- **短期知识**: 场景特异性判别特征 (弱约束 0.5x)

### 2. 自适应知识融合 ⭐⭐
根据特征相似度动态调整长短期知识权重:
```python
weight_long = sim_long / (sim_long + sim_short)
weight_short = 1 - weight_long
```

### 3. 域混淆损失 ⭐
增强长期知识的跨域泛化能力:
```python
loss_domain = -similarity_matrix.mean() * 0.1
```

## 训练流程

### 第一个数据集 (Market1501)
```
训练基础模型 (80 epochs)
    ↓
保存模型和特征
```

### 第二个数据集及以后 (DukeMTMC, CUHK03, MSMT17)

#### 阶段 1: 知识解耦 (5 epochs)
```
旧模型 Model_old
    ↓
冻结 backbone
    ↓
训练长期模型 (域不变特征)
  - 特征重构 + 域混淆损失
    ↓
训练短期模型 (场景特异特征)
  - 精确重构 + 分类 + 正则化
    ↓
输出: Model_long, Model_short
```

#### 阶段 2: 新任务训练 (60 epochs)
```
每个批次:
  1. 当前模型前向传播
  2. 计算基础损失 (CE + Triplet + Center)
  3. 提取长短期知识 (无梯度)
  4. 特征转换
  5. 计算自适应权重
  6. 计算 LSTKC 损失
  7. 总损失 = 基础 + BI-C2R + LSTKC
  8. 反向传播优化
```

## 已解决的问题

### ✓ OpenCV 依赖缺失
- **问题**: `ImportError: libGL.so.1: cannot open shared object file`
- **解决**: 安装 `libgl1-mesa-glx` 和 `libglib2.0-0`

### ✓ yacs 模块缺失
- **问题**: `ModuleNotFoundError: No module named 'yacs'`
- **解决**: `pip install yacs`

### ✓ 参数格式错误
- **问题**: `--milestones 40,70` 格式错误
- **解决**: 改为 `--milestones 40 70` (空格分隔)

### ✓ 参数名称不匹配
- **问题**: `--batch_size` vs `-b`
- **解决**: 统一使用短横线格式 `-b` 或 `--batch-size`

### ✓ LSTKC++ 参数未定义
- **问题**: 新增参数未在 argparse 中定义
- **解决**: 在 `continual_train.py` 中添加所有 LSTKC++ 参数
