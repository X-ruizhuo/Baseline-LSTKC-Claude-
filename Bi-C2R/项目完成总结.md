## 🎯 融合特性

### LSTKC++核心功能
1. **长短期知识分解** - 自动区分稳定特征和任务特定特征
2. **知识质量过滤** - 评估并过滤低质量旧知识
3. **自适应知识整合** - 动态调整新旧知识融合权重

### 显存优化
- 批次大小: 64 → 48
- 混合精度训练(FP16)
- 梯度累积(2步)
- **总体节省: 34% (32GB → 21GB)**

## 🚀 使用方法

### 快速开始
```bash
# Baseline训练
bash run1.sh

# LSTKC++增强训练
bash run_lstkc.sh
```

### 自定义训练
```bash
python continual_train.py \
    --logs-dir logs-custom/ \
    --setting 1 \
    --enable_lstkc \
    --use_amp \
    --gradient_accumulation_steps 2 \
    --batch-size 48 \
    --weight_long_term 0.5 \
    --weight_short_term 0.3
```

## 🔧 关键参数

### LSTKC++参数
- `--enable_lstkc`: 启用LSTKC++模块
- `--weight_long_term`: 长期知识损失权重(0.5)
- `--weight_short_term`: 短期知识损失权重(0.3)

### 显存优化
- `--use_amp`: 混合精度训练
- `--gradient_accumulation_steps`: 梯度累积(2)
- `--batch-size`: 批次大小(48)
