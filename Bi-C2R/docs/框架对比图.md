```
┌─────────────────────────────────────────────────────────────────┐
│                    Bi-C2R Baseline 框架                          │
└─────────────────────────────────────────────────────────────────┘

输入: 新任务数据 (Task t)
  │
  ├─────────────────────────────────────────────────────────────┐
  │                                                             │
  ▼                                                             ▼
┌──────────────────┐                                  ┌──────────────────┐
│  当前模型 (New)  │                                  │  旧模型 (Old)    │
│   ResNet-50      │                                  │   ResNet-50      │
│   [可训练]       │                                  │   [冻结状态]     │
└────────┬─────────┘                                  └────────┬─────────┘
         │                                                     │
         │ 提取新数据特征                                      │ 提取新数据特征
         │                                                     │ (用于对比和蒸馏)
         ▼                                                     ▼
    f_new (2048维)                                        f_old (2048维)
         │                                                     │
         │                                                     │
         ├─────────────────────┬───────────────────────────────┤
         │                     │                               │
         ▼                     ▼                               ▼
┌─────────────────┐   ┌──────────────────┐         ┌──────────────────┐
│  分类损失       │   │  三元组损失      │         │  KL散度约束      │
│  (Loss_ce)      │   │  (Loss_tp)       │         │  (Divergence)    │
└─────────────────┘   └──────────────────┘         └──────────────────┘
                                                             │
         ┌───────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────────┐
│              双向转换网络 (Bidirectional TransNet)               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  f_old ──────► TransNet_1 ──────► f_old→new                   │
│                    │                    │                       │
│                    │                    ├──► Loss_trans (MSE)  │
│                    │                    │                       │
│  f_new ──────► TransNet_2 ──────► f_new→old                   │
│                                         │                       │
│                                         ├──► Loss_anti (KL)    │
│                                         │                       │
│                                         ├──► Loss_discri (KD)  │
│                                         │                       │
│                                         └──► Loss_transx (Cos) │
└─────────────────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────────────┐
│                    自适应Alpha融合                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  计算亲和矩阵差异:                                               │
│  Affinity_new = softmax(f_new @ f_new.T)                       │
│  Affinity_old = softmax(f_old @ f_old.T)                       │
│  Difference = |Affinity_new - Affinity_old|                    │
│  alpha = 1 - Difference                                         │
│                                                                 │
│  最终模型 = alpha * model_new + (1-alpha) * model_old          │
└─────────────────────────────────────────────────────────────────┘
         │
         ▼
    输出: 融合后的模型

【核心特点】
✓ 双向转换保持特征兼容性
✓ 基于亲和矩阵差异的自适应融合
✓ 多损失函数协同优化
✗ 新旧知识混合，无区分机制
✗ 无知识质量评估
✗ 可能存在负迁移问题
```

         │                                                     │

         │                                                     │

         │                     │                               │

┌─────────────────┐   ┌──────────────────┐         ┌──────────────────┐

│  (Loss_ce)      │   │  (Loss_tp)       │         │  (Divergence)    │
└─────────────────┘   └──────────────────┘         └──────────────────┘
                                                             │
         ┌───────────────────────────────────────────────────┘
         │
         ▼
┌═════════════════════════════════════════════════════════════════┐
║          【新增】LSTKC++ 知识分解与过滤模块                      ║
╠═════════════════════════════════════════════════════════════════╣
║                                                                 ║
║  ┌─────────────────────────────────────────────────────────┐   ║
║  │  步骤1: 知识分解 (Knowledge Decomposition)              │   ║
║  ├─────────────────────────────────────────────────────────┤   ║
║  │                                                         │   ║
║  │  f_old ──┬──► long_term_extractor ──► LT_old (通用)   │   ║
║  │          │                                              │   ║
║  │          └──► short_term_extractor ──► ST_old (专用)   │   ║
║  │                         │                               │   ║
║  │                         ├──► gate_network ──► w_old    │   ║
║  │                         │                               │   ║
║  │          融合: f_old_decomp = w_old * LT_old +         │   ║
║  │                               (1-w_old) * ST_old       │   ║
║  │                                                         │   ║
║  │  f_new ──┬──► long_term_extractor ──► LT_new (通用)   │   ║
║  │          │                                              │   ║
║  │          └──► short_term_extractor ──► ST_new (专用)   │   ║
║  │                         │                               │   ║
║  │                         ├──► gate_network ──► w_new    │   ║
║  │                         │                               │   ║
║  │          融合: f_new_decomp = w_new * LT_new +         │   ║
║  │                               (1-w_new) * ST_new       │   ║
║  └─────────────────────────────────────────────────────────┘   ║
║                         │                                       ║
║                         ▼                                       ║
║  ┌─────────────────────────────────────────────────────────┐   ║
║  │  步骤2: 知识过滤 (Knowledge Filter)                     │   ║
║  ├─────────────────────────────────────────────────────────┤   ║
║  │                                                         │   ║
║  │  输入: [f_old, f_new] (4096维)                         │   ║
║  │    │                                                    │   ║
║  │    ├──► Quality Estimator Network                      │   ║
║  │    │      (Linear 4096→512→128→1)                      │   ║
║  │    │                                                    │   ║
║  │    └──► quality_score (0-1)                            │   ║
║  │              │                                          │   ║
║  │              ├─ if score > 0.6: 保留旧知识             │   ║
║  │              └─ if score ≤ 0.6: 抑制旧知识 (×0.5)      │   ║
║  │                                                         │   ║
║  │  输出: quality_weight (用于加权损失)                    │   ║
║  └─────────────────────────────────────────────────────────┘   ║
║                         │                                       ║
║                         ▼                                       ║
║  ┌─────────────────────────────────────────────────────────┐   ║

║  │                                                         │   ║
║  │  【新增】长期知识保持损失:                              │   ║
║  │  Loss_long = MSE(LT_new, LT_old.detach())              │   ║
║  │  权重: 0.5                                              │   ║
║  │                                                         │   ║
║  │  【新增】短期知识适应损失:                              │   ║
║  │  Loss_short = MSE(ST_new, f_new)                       │   ║
║  │  权重: 0.3                                              │   ║
║  │                                                         │   ║
║  │  【增强】质量加权的转换损失:                            │   ║
║  │  Loss_trans_weighted = quality_weight * Loss_trans     │   ║
║  │                                                         │   ║
║  └─────────────────────────────────────────────────────────┘   ║
╚═════════════════════════════════════════════════════════════════╝

┌─────────────────────────────────────────────────────────────────┐

│                                                                 │
│  f_old_decomp ──► TransNet_1 ──► f_old→new                     │
│                      │                │                         │
│                      │                ├──► Loss_trans (MSE)    │
│                      │                │     × quality_weight   │
│                      │                │                         │
│  f_new_decomp ──► TransNet_2 ──► f_new→old                     │
│                                       │                         │
│                                       ├──► Loss_anti (KL)      │
│                                       │                         │
│                                       ├──► Loss_discri (KD)    │
│                                       │                         │
│                                       └──► Loss_transx (Cos)   │
└─────────────────────────────────────────────────────────────────┘
         │
         ▼
┌═════════════════════════════════════════════════════════════════┐
║         【增强】自适应Alpha融合 + 长短期稳定性评估               ║
╠═════════════════════════════════════════════════════════════════╣
║                                                                 ║
║  基础Alpha计算 (继承Baseline):                                  ║
║  alpha_base = 1 - |Affinity_new - Affinity_old|                ║
║                                                                 ║

║                                                                 ║

║  增强Alpha计算:                                                 ║
║  alpha_enhanced = (1 - affinity_diff) * stability +            ║
║                   affinity_diff * (1 - importance)             ║
║  alpha_final = clamp(alpha_enhanced, 0.3, 0.9)                 ║
║                                                                 ║
║  最终模型 = alpha_final * model_new + (1-alpha_final) * model_old ║
╚═════════════════════════════════════════════════════════════════╝
         │
         ▼
┌═════════════════════════════════════════════════════════════════┐
║              【新增】显存优化模块                                ║
╠═════════════════════════════════════════════════════════════════╣
║  • 混合精度训练 (FP16): 节省50%显存                             ║
║  • 梯度累积 (2步): 等效更大batch size                           ║

## 三、关键模块对比表

| 模块 | Baseline (Bi-C2R) | 本项目 (Bi-C2R + LSTKC++) |
|------|-------------------|---------------------------|
| **特征提取** | ResNet-50 | ResNet-50 (相同) |
| **知识管理** | 新旧特征混合 | 显式分解为长期/短期知识 |
| **知识分解** | ❌ 无 | ✅ KnowledgeDecomposition模块 |
| **知识过滤** | ❌ 无 | ✅ KnowledgeFilter模块 (阈值0.6) |
| **转换网络** | TransNet (简单MLP) | LSTKC-Enhanced TransNet |
| **Alpha计算** | 基于亲和矩阵差异 | 增强版 (考虑长短期稳定性) |
| **损失函数** | 6个损失 | 8个损失 (+长期+短期) |
| **防遗忘策略** | KL散度约束 | KL散度 + 知识分解 + 质量过滤 |
| **显存优化** | ❌ 无 | ✅ FP16 + 梯度累积 |
| **Batch Size** | 64 | 48 (可调) |
| **显存占用** | ~32GB | ~21GB (-34%) |
| **参数量** | ~25M | ~28M (+知识分解网络) |

## 四、数据流对比

### Baseline数据流:
```
输入图像 → ResNet → f_new/f_old → TransNet → 损失计算 → 反向传播
```

### 本项目数据流:
```
输入图像 → ResNet → f_new/f_old 
         ↓
    知识分解 (LT/ST)
         ↓
    知识过滤 (quality_score)
         ↓
    TransNet (分解后特征)
         ↓
    增强损失计算 (8个损失)
         ↓
    混合精度反向传播 (FP16)
         ↓
    梯度累积 (2步)
```
